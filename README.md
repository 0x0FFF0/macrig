# macrig Installation

## Quick Install

```bash
curl -sL bit.ly/macrig | bash
```

Or with auto-confirmation:
```bash
curl -sL https://bit.ly/macrig | bash -s -- -y
```

## Usage

After installation, run:
```bash
syspolicy
```

Or navigate to project directory:
```bash
cd ~/.local/share/src/syspolicy
python3 syspolicy.py
```
